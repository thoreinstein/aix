## aix init

Initialize aix configuration

### Synopsis

Bootstrap aix configuration with automatic platform detection.

Creates ~/.config/aix/config.yaml with detected AI coding platforms.
Platforms are detected by checking if their config directories exist.

```
aix init [flags]
```

### Examples

```
  # Initialize with interactive prompts
  aix init

  # Initialize non-interactively, accepting defaults
  aix init --yes

  # Initialize for specific platforms
  aix init --platforms claude,opencode

  # Force overwrite existing configuration
  aix init --force

  See Also: aix config, aix doctor
```

### Options

```
  -f, --force              Overwrite existing configuration
  -h, --help               help for init
      --platforms string   Comma-separated list of platforms to configure (overrides auto-detection)
  -y, --yes                Non-interactive mode, accept all defaults
```
