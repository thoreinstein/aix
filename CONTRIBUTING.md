# Contributing to aix

Thank you for your interest in contributing to `aix`!

## Getting Started

1.  Fork the repository.
2.  Clone your fork.
3.  Create a branch for your feature or bug fix.

## Development

`aix` is written in Go. Ensure you have Go 1.22+ installed.

```bash
go build ./cmd/aix
```

## Pull Requests

Please ensure your code passes all tests and linters before submitting a PR.
